# NODE_BLOG_DEMO
这是用node+express+mysql做的一个博客，分为博客网站和后台管理系统
----------------------------------------------------------------------------------------------------

blog_manage:
该项目是博客系统的后台管理系统，用来发布博文及维护博文的一些基本操作；
blog_new_node:
该项目是博客系统的前台展示平台，包括博文分类展示，集赞，文字评论等功能；
blog_new:
该项目是博客的前端静态页面；

----------------------------------------------------------------------------------------------------

该项目可以直接运行，只是数据库的话要改一下url，以学习为目的的blog demo
